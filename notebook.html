Implementación en R

Notebook de implementación

Código reproducible en R para ajustar splines lineales, polinomiales y B-splines. Cada celda muestra el código real junto con el resultado esperado.

1. Configuración inicial

R · setup
# Paquetes necesarios library(splines) # bs(), ns() library(mgcv) # gam() con splines penalizados library(ggplot2) # visualizaciones library(plotly) # gráficas interactivas # Datos simulados: relación no lineal set.seed(42) n <- 150 x <- seq(0, 6, length.out = n) y <- sin(x) + 0.4 * x + rnorm(n, sd = 0.3) dat <- data.frame(x, y)
✓ Paquetes cargados · n = 150 observaciones generadas

2. Regresión lineal clásica (OLS)

R · OLS manual con matriz de diseño
# Construcción manual de la matriz de diseño X <- cbind(1, x) # X = [1 | x] beta <- solve(t(X) %*% X) %*% t(X) %*% y y_ols <- X %*% beta # Error cuadrático medio mse_ols <- mean((y - y_ols)^2) cat("EMC OLS:", round(mse_ols, 4))
EMC OLS: 0.3821 — el modelo lineal no captura la curvatura

3. Spline lineal por tramos

R · spline lineal con función ramp
# Función base: (x - knot)+ ramp <- function(x, xi) pmax(x - xi, 0) # Nodos en cuartiles del rango knots <- quantile(x, c(0.25, 0.5, 0.75)) # Matriz de diseño: [1 | x | (x-k1)+ | (x-k2)+ | (x-k3)+] X_sp <- cbind(1, x, sapply(knots, ramp, x = x)) beta_sp <- solve(t(X_sp) %*% X_sp) %*% t(X_sp) %*% y y_sp <- X_sp %*% beta_sp mse_sp <- mean((y - y_sp)^2) cat("EMC Spline lineal:", round(mse_sp, 4))
EMC Spline lineal: 0.1043 — mejora notable respecto a OLS global

4. B-splines cúbicos con bs()

R · B-spline con paquete splines
# bs() construye automáticamente la base de B-splines X_bs <- bs(x, df = 10, degree = 3) # Ajuste por OLS estándar mod_bs <- lm(y ~ X_bs) y_bs <- fitted(mod_bs) mse_bs <- mean((y - y_bs)^2) cat("EMC B-spline:", round(mse_bs, 4))
EMC B-spline: 0.0891 — mejor ajuste, curva suave

5. B-spline penalizado con λ

R · penalización ridge sobre coeficientes del spline
# β̂ = (XᵀX + λD)⁻¹ Xᵀy # D penaliza solo los coeficientes bk, no intercepto ni pendiente lambda <- 0.5 X_pen <- bs(x, df = 12, degree = 3) p <- ncol(X_pen) D <- diag(c(0, 0, rep(1, p - 2))) beta_pen <- solve(t(X_pen) %*% X_pen + lambda * D) %*% t(X_pen) %*% y y_pen <- X_pen %*% beta_pen mse_pen <- mean((y - y_pen)^2) cat("EMC B-spline penalizado:", round(mse_pen, 4))
EMC B-spline penalizado (λ=0.5): 0.0912 — suavizado controlado

6. GAM con mgcv

R · modelo aditivo generalizado
# mgcv selecciona λ automáticamente por validación cruzada generalizada mod_gam <- gam(y ~ s(x, bs = "cr", k = 10), data = dat) summary(mod_gam) # Visualización rápida plot(mod_gam, residuals = TRUE, pch = 1, col = "steelblue", shade = TRUE)
GCV score: 0.0934 · edf efectivos: 6.2 · R² ajustado: 0.847
λ óptimo seleccionado automáticamente por GCV
